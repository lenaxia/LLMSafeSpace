## Critical Design Constraint: DEK/Session Requirement

**The encryption model requires an active DEK session for any operation that touches secret plaintext.** DEK sessions are created at login (JWT issuance) and cached in Redis with TTL matching the JWT lifetime.

**Impact on MCP/SDK callers:**

| Operation | Needs DEK? | Works with API key? | Works with JWT? |
|-----------|-----------|--------------------|-----------------| 
| `credential_list` | No | ‚úÖ | ‚úÖ |
| `credential_delete` | No | ‚úÖ | ‚úÖ |
| `credential_bind` | No (bind only; push fails silently) | ‚öÝÔ∏è Partial | ‚úÖ |
| `credential_create` | Yes (encrypts value) | ‚ùå | ‚úÖ |
| `model_list` | No (queries pod) | ‚úÖ | ‚úÖ |
| `model_set` | Yes (decrypt + re-encrypt) | ‚ùå | ‚úÖ |

**Mitigation for MCP/SDK users:** The MCP client must authenticate with a JWT (login first, then use the token). The `HTTPClient` supports `Authorization: Bearer <jwt>` headers which carry the jti claim for session lookup.

**Future fix (out of scope here):** Add a "session start" API for API key users that unlocks the DEK and returns a session token. This is an Epic 10 follow-up.

**For this story:** Accept the constraint. Document it. MCP tools that need DEK will return a clear error ("active session required ‚Äî authenticate with JWT") rather than failing silently.

### Task 4: Model tier annotation

**File:** `api/internal/handlers/models.go`

**Change:** Instead of raw pass-through (`c.Data(200, "application/json", body)`), parse the opencode response, annotate each model with:

```go
type AnnotatedModel struct {
    // All fields from opencode's response (pass through as raw JSON)
    Raw           json.RawMessage `json:"raw"`
    // Our annotations
    ID            string `json:"id"`
    ProviderID    string `json:"providerID"`
    Name          string `json:"name"`
    Tier          string `json:"tier"`          // "free" or "paid"
    ProxyRequired bool   `json:"proxyRequired"` // true for free models (Epic 26 future)
}
```

**Detection logic:**
```go
func annotateModelTier(providerID string, cost []CostEntry) string {
    if providerID == "opencode" {
        // All opencode-provider models with zero input cost are free
        allFree := true
        for _, c := range cost {
            if c.Input > 0 { allFree = false; break }
        }
        if allFree { return "free" }
    }
    return "paid"
}
```

**Issue:** opencode's model response is complex (nested structs with variants, costs, capabilities). We don't want to maintain a full mirror of their types.

**Solution:** Parse only the fields we need for annotation (`id`, `providerID`, `name`, `cost[].input`), pass through the full object as raw JSON. Response format:

```json
[
  {
    "id": "opencode/some-free-model",
    "providerID": "opencode",
    "name": "Some Model",
    "tier": "free",
    "proxyRequired": false,
    "details": { /* full opencode ModelV2.Info object */ }
  }
]
```

**`credential_create` handler logic:**

```go
func (h *handlers) credentialCreate(ctx, req) {
    args := req.GetArguments()
    provider, _ := args["provider"].(string)
    apiKey, _ := args["api_key"].(string)
    
    if provider == "" || apiKey == "" {
        return mcp.NewToolResultError("provider and api_key are required"), nil
    }
    
    resp, err := h.client.CreateCredential(ctx, CreateCredentialReq{
        Name:        strArg(args, "name"),
        Provider:    provider,
        APIKey:      apiKey,
        BaseURL:     strArg(args, "base_url"),
        Default:     strArg(args, "default_model"),
        WorkspaceID: strArg(args, "workspace_id"),
    })
    if err != nil {
        return mcp.NewToolResultError(fmt.Sprintf("failed: %v", err)), nil
    }
    
    out, _ := json.Marshal(resp)
    return mcp.NewToolResultText(string(out)), nil
}
```

### Task 6: HTTPClient implementation for new methods

**File:** `pkg/mcp/client.go`

**Implementations:** All use `c.doJSON()` pattern (same as existing `CreateWorkspace`, etc.)

```go
func (c *HTTPClient) CreateCredential(ctx context.Context, req CreateCredentialReq) (*CredentialResp, error) {
    providerData := map[string]any{
        "provider": req.Provider,
        "apiKey":   req.APIKey,
    }
    if req.BaseURL != "" { providerData["baseURL"] = req.BaseURL }
    if req.Default != "" { providerData["default"] = req.Default }
    valueJSON, _ := json.Marshal(providerData)
    
    body := map[string]any{
        "name":     req.Name,
        "type":     "llm-provider",
        "value":    string(valueJSON),
        "metadata": map[string]string{},
    }
    if req.Name == "" { body["name"] = req.Provider }
    
    var resp CredentialResp
    if err := c.doJSON(ctx, http.MethodPost, "/api/v1/secrets", body, &resp); err != nil {
        return nil, err
    }
    return &resp, nil
}
```

The other methods are trivial HTTP wrappers.
