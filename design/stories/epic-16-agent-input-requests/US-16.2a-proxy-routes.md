# US-16.2a: Question/Permission Proxy Routes

**Epic:** 16 - Agent Input Requests
**Priority:** Critical
**Depends on:** US-16.1
**Layer:** Backend

## User Story

As a frontend or MCP client, I need REST endpoints to list pending questions/permissions and submit replies, so that I can interact with the agent's input request system through the LLMSafeSpace API.

## Acceptance Criteria

- [ ] 5 new routes registered in `registerProxyRoutes`
- [ ] All handlers live in a NEW file `api/internal/handlers/proxy_input.go`
- [ ] All routes go through existing auth middleware (same as session proxy routes)
- [ ] All routes use `proxyToWorkspace` pattern (workspace must be Active, pod IP resolved)
- [ ] Request body is forwarded as-is (no transformation)
- [ ] Response is forwarded as-is (no filtering, no buffering)
- [ ] `requestID` parameter is validated before hitting the pod
- [ ] Invalid `requestID` returns 400
- [ ] `ProxyHandler` gains a `dialect agent.Dialect` field, injected via `NewProxyHandler`

## Technical Details

### New File: `api/internal/handlers/proxy_input.go`

All question/permission handlers live here from the start — they never go into the 900-line `proxy.go`.

```go
package handlers

import (
    "net/http"
    "regexp"

    "github.com/gin-gonic/gin"
    "github.com/lenaxia/llmsafespace/pkg/agent"
)

var (
    questionIDPattern   = regexp.MustCompile(`^que_[a-zA-Z0-9]+$`)
    permissionIDPattern = regexp.MustCompile(`^per_[a-zA-Z0-9]+$`)
)

func (h *ProxyHandler) ListQuestions(c *gin.Context) {
    h.proxyToWorkspace(c, h.dialect.QuestionListPath(), false, "")
}

func (h *ProxyHandler) QuestionReply(c *gin.Context) {
    requestID := c.Param("requestID")
    if !questionIDPattern.MatchString(requestID) {
        c.JSON(http.StatusBadRequest, gin.H{"error": "invalid question request ID format"})
        return
    }
    h.proxyToWorkspace(c, h.dialect.QuestionReplyPath(requestID), false, "")
}

func (h *ProxyHandler) QuestionReject(c *gin.Context) {
    requestID := c.Param("requestID")
    if !questionIDPattern.MatchString(requestID) {
        c.JSON(http.StatusBadRequest, gin.H{"error": "invalid question request ID format"})
        return
    }
    h.proxyToWorkspace(c, h.dialect.QuestionRejectPath(requestID), false, "")
}

func (h *ProxyHandler) ListPermissions(c *gin.Context) {
    h.proxyToWorkspace(c, h.dialect.PermissionListPath(), false, "")
}

func (h *ProxyHandler) PermissionReply(c *gin.Context) {
    requestID := c.Param("requestID")
    if !permissionIDPattern.MatchString(requestID) {
        c.JSON(http.StatusBadRequest, gin.H{"error": "invalid permission request ID format"})
        return
    }
    h.proxyToWorkspace(c, h.dialect.PermissionReplyPath(requestID), false, "")
}
```

### Route Registration

In `api/internal/server/router.go`, add to `registerProxyRoutes`:

```go
// Question/Permission input request routes
rg.GET("/:id/question", proxyHandler.ListQuestions)
rg.POST("/:id/question/:requestID/reply", proxyHandler.QuestionReply)
rg.POST("/:id/question/:requestID/reject", proxyHandler.QuestionReject)
rg.GET("/:id/permission", proxyHandler.ListPermissions)
rg.POST("/:id/permission/:requestID/reply", proxyHandler.PermissionReply)
```
