# US-16.3: Normalized Event Emission

**Epic:** 16 - Agent Input Requests
**Priority:** Critical
**Depends on:** US-16.1
**Layer:** Backend

## User Story

As a frontend developer, I need the backend to emit stable, agent-agnostic SSE events when a question or permission is pending/resolved, so that the frontend can render prompts without parsing raw opencode event internals.

## Acceptance Criteria

- [ ] When `onRawEvent` receives a `question.asked` event, it publishes an additional `WorkspaceSSEEvent` with `Type: "agent.question"` and `Data` set to the normalized `QuestionRequest`
- [ ] When `onRawEvent` receives a `question.replied` or `question.rejected` event, it publishes `WorkspaceSSEEvent` with `Type: "agent.question.resolved"` and `Data` containing `{requestID, sessionID}`
- [ ] When `onRawEvent` receives a `permission.asked` event, it publishes `WorkspaceSSEEvent` with `Type: "agent.permission"` and `Data` set to the normalized `PermissionRequest`
- [ ] When `onRawEvent` receives a `permission.replied` event, it publishes `WorkspaceSSEEvent` with `Type: "agent.permission.resolved"` and `Data` containing `{requestID, sessionID, reply}`
- [ ] The raw `opencode.event` is STILL published (no regression — streaming/tool rendering depends on it)
- [ ] Parse errors are logged at Warn level and do NOT crash the SSE stream or suppress the raw event
- [ ] The normalized events use the types from `pkg/agent/types.go` (not opencode-specific shapes)

## Technical Details

### Modified `onRawEvent` in `proxy.go`

```go
func (h *ProxyHandler) onRawEvent(workspaceID, eventType, rawData string) {
    if h.broker == nil {
        return
    }

    // Always publish raw event (existing behavior — no regression)
    var parsed interface{}
    json.Unmarshal([]byte(rawData), &parsed)
    h.broker.Publish(workspaceID, WorkspaceSSEEvent{
        Type:      "opencode.event",
        EventType: eventType,
        Data:      parsed,
    })

    // Existing: persist session title
    if eventType == "session.updated" && h.sessionIndex != nil {
        h.persistTitleFromEvent(workspaceID, rawData)
    }

    // NEW: Emit normalized input request events
    h.emitNormalizedInputEvent(workspaceID, eventType, rawData)
}

func (h *ProxyHandler) emitNormalizedInputEvent(workspaceID, eventType, rawData string) {
    // Extract properties from the event envelope
    var envelope struct {
        Properties json.RawMessage `json:"properties"`
    }
    if json.Unmarshal([]byte(rawData), &envelope) != nil || len(envelope.Properties) == 0 {
        return
    }

    if h.dialect.IsQuestionAsked(eventType) {
        req, err := h.dialect.ParseQuestionRequest(eventType, envelope.Properties)
        if err != nil {
            h.logger.Warn("Failed to parse question event", "error", err, "workspaceID", workspaceID)
            return
        }
        h.broker.Publish(workspaceID, WorkspaceSSEEvent{
            Type: "agent.question",
            Data: req,
        })
    } else if h.dialect.IsQuestionResolved(eventType) {
        h.broker.Publish(workspaceID, WorkspaceSSEEvent{
            Type: "agent.question.resolved",
            Data: extractResolutionData(envelope.Properties),
        })
    } else if h.dialect.IsPermissionAsked(eventType) {
        req, err := h.dialect.ParsePermissionRequest(eventType, envelope.Properties)
        if err != nil {
            h.logger.Warn("Failed to parse permission event", "error", err, "workspaceID", workspaceID)
            return
        }
        h.broker.Publish(workspaceID, WorkspaceSSEEvent{
            Type: "agent.permission",
            Data: req,
        })
    } else if h.dialect.IsPermissionResolved(eventType) {
        h.broker.Publish(workspaceID, WorkspaceSSEEvent{
            Type: "agent.permission.resolved",
            Data: extractResolutionData(envelope.Properties),
        })
    }
}
```

### New Event Types Added to `WorkspaceSSEEvent` Documentation

```go
// Event types:
//   - "workspace.phase":            workspace phase changed
//   - "session.status":             session idle/busy notification
//   - "opencode.event":             raw event forwarded from agent
//   - "agent.question":             agent is asking user a question (Data: *agent.QuestionRequest)
//   - "agent.question.resolved":    question was answered or dismissed (Data: {requestID, sessionID})
//   - "agent.permission":           agent needs permission approval (Data: *agent.PermissionRequest)
//   - "agent.permission.resolved":  permission was approved or denied (Data: {requestID, sessionID, reply})
```

### Wire Format (what the browser receives)

```json
{"type":"agent.question","data":{"id":"que_abc","session_id":"ses_xyz","questions":[...],"tool":{...}}}
```

```json
{"type":"agent.question.resolved","data":{"request_id":"que_abc","session_id":"ses_xyz"}}
```

```json
{"type":"agent.permission","data":{"id":"per_abc","session_id":"ses_xyz","permission":"shell","patterns":["rm -rf /tmp"]}}
```
