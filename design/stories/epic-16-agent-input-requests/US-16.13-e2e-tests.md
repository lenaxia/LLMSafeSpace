# US-16.13: E2E Integration Tests

**Epic:** 16 - Agent Input Requests
**Priority:** High
**Depends on:** US-16.1–16.12
**Layer:** Both

## User Story

As a developer, I need end-to-end tests that exercise the full question/permission flow from prompt to resolution, so that I can be confident the system works as a whole and catch integration regressions.

## Acceptance Criteria

- [ ] Backend integration test: prompt → question.asked SSE → reply API → session status idle
- [ ] Backend integration test: prompt → permission.asked SSE → auto-approve (autoApprovePermissions=true) → session status idle
- [ ] Backend integration test: list pending questions returns correct data
- [ ] MCP integration test: session_message → returns question → session_question_reply → session_history has answer
- [ ] Frontend E2E test (Playwright or similar): send message → question prompt appears → click answer → prompt disappears → response renders
- [ ] Frontend E2E test: page refresh while question pending → prompt re-appears

### Test Infrastructure Requirements

- Backend tests need either a real opencode pod OR a mock HTTP server that simulates the question/permission API
- The mock must: accept `POST /session/:id/prompt_async`, emit `question.asked` on `/event` SSE, accept `POST /question/:id/reply`, emit `question.replied` + `session.status idle`
- Frontend E2E tests need the full stack running (API + pod) OR MSW (Mock Service Worker) intercepting API calls

### Mock Server for Backend Tests

```go
// testutil/mock_opencode.go
type MockOpencode struct {
    pendingQuestions []QuestionRequest
    pendingPerms    []PermissionRequest
    eventCh         chan string // SSE events to emit
}

func (m *MockOpencode) ServeHTTP(w http.ResponseWriter, r *http.Request) {
    switch {
    case r.URL.Path == "/question" && r.Method == "GET":
        json.NewEncoder(w).Encode(m.pendingQuestions)
    case strings.HasSuffix(r.URL.Path, "/reply") && strings.Contains(r.URL.Path, "/question/"):
        // Remove from pending, emit replied event
        w.Write([]byte("true"))
    case r.URL.Path == "/event" && r.Method == "GET":
        // SSE stream
        flusher := w.(http.Flusher)
        w.Header().Set("Content-Type", "text/event-stream")
        for evt := range m.eventCh {
            fmt.Fprintf(w, "data: %s\n\n", evt)
            flusher.Flush()
        }
    }
}
```
