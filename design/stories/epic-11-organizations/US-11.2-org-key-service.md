# US-11.2: Org Key Service

**Epic:** 11 (Organizations)
**Depends On:** US-11.1 (schema)
**Estimated Effort:** ~6 hours

---

## Goal

Implement `OrgKeyService` — a new struct in `pkg/secrets/` that manages the org DEK lifecycle. Parallel to `KeyService` (which handles user DEKs) but purpose-built for the per-admin-wrapped org DEK model described in the README.

This story does NOT wire `OrgKeyService` into the login path (that is US-11.3) or the `decryptBinding` path (US-11.7). It builds and tests the service in isolation.

## Design

### `OrgKeyStore` interface

```go
// pkg/secrets/org_key_store.go

// OrgKeyMemberRecord is one row in org_key_members.
type OrgKeyMemberRecord struct {
    OrgID      string
    UserID     string
    WrappedDEK []byte
    KeyVersion int
}

// OrgKeyStore abstracts DB operations for org key material.
type OrgKeyStore interface {
    // GetOrgKeyMember returns the wrapped DEK for (orgID, userID). Returns nil, nil if not found.
    GetOrgKeyMember(ctx context.Context, orgID, userID string) (*OrgKeyMemberRecord, error)
    // UpsertOrgKeyMember inserts or updates a wrapped DEK row for (orgID, userID).
    // The org_memberships row for (orgID, userID) must already exist (composite FK).
    UpsertOrgKeyMember(ctx context.Context, record *OrgKeyMemberRecord) error
    // DeleteOrgKeyMember removes the wrapped DEK for (orgID, userID).
    // In practice this is handled by CASCADE from org_memberships deletion, but
    // this method exists for explicit use in RotateOrgDEK bulk-delete.
    DeleteOrgKeyMember(ctx context.Context, orgID, userID string) error
    // ListOrgKeyMembers returns all wrapped DEK rows for an org (used by RotateOrgDEK).
    ListOrgKeyMembers(ctx context.Context, orgID string) ([]*OrgKeyMemberRecord, error)
    // DeleteAllOrgKeyMembers deletes every org_key_members row for an org in one operation.
    // Used by RotateOrgDEK to atomically clear all admin key rows before inserting the
    // rotating admin's new row. Must be called within the same pgx.Tx as the insert.
    DeleteAllOrgKeyMembers(ctx context.Context, orgID string) error
    // GetOrgKeyMembersForUser returns all org_key_members rows for a given user across
    // all orgs. Used by the login path to batch-fetch all org DEK records in one query
    // instead of one query per org. Returns empty slice (not error) if user has no rows.
    GetOrgKeyMembersForUser(ctx context.Context, userID string) ([]*OrgKeyMemberRecord, error)
    // GetUserSalt returns the salt from user_keys for a given userID.
    // Needed to derive KEK during key handshake and rotation without requiring the caller
    // to reach into the user key service.
    GetUserSalt(ctx context.Context, userID string) ([]byte, error)
}
```

### `OrgKeyService` struct

```go
// pkg/secrets/org_key_service.go

const orgKEKInfo = "llmsafespace-org-kek"

// OrgKeyService manages org DEK lifecycle.
type OrgKeyService struct {
    store  OrgKeyStore
    cache  DEKCache   // same DEKCache interface used by KeyService
    logger pkginterfaces.LoggerInterface
}

func NewOrgKeyService(store OrgKeyStore, cache DEKCache) *OrgKeyService

// OrgCacheKey returns the DEK cache key for an org.
// Convention: "org:<orgID>" — namespaced to prevent collision with session IDs (JWT jti = UUID).
func OrgCacheKey(orgID string) string { return "org:" + orgID }
```

### Methods

#### `InitializeOrgKeys`

Called once when an org is created, by the founding admin. The founding admin's plaintext password must be provided (available at org-creation request time since the request is authenticated and the user just logged in).

```go
// InitializeOrgKeys generates the org DEK and wraps it with the founding admin's KEK.
// adminPassword is the founding admin's plaintext password, provided in the create-org
// request body (only required at creation time; never stored).
// adminUserID is the founding admin's user ID.
// Returns the org DEK in memory; caller caches it under OrgCacheKey(orgID) for the session.
func (s *OrgKeyService) InitializeOrgKeys(ctx context.Context, orgID, adminUserID string, adminPassword []byte) (orgDEK []byte, err error)
```

Implementation:
1. `orgDEK, err := GenerateDEK()` — note: `GenerateDEK` returns `([]byte, error)`; handle the error
2. `adminSalt, err = store.GetUserSalt(ctx, adminUserID)` — salt from `user_keys.salt`
3. `adminKEK, err = DeriveKEK(adminPassword, adminSalt, orgKEKInfo)`
4. `wrappedDEK, err = WrapDEK(adminKEK, orgDEK)`
5. `store.UpsertOrgKeyMember(ctx, &OrgKeyMemberRecord{OrgID: orgID, UserID: adminUserID, WrappedDEK: wrappedDEK, KeyVersion: 1})`
6. Return `orgDEK` (caller caches it; this function does not cache)

#### `UnlockOrgDEK`

Called during login for each org where the user is an admin. **Not called directly by the login path** — the login path calls `UnlockAllOrgDEKs` (below) which does a single batch DB query.

```go
// UnlockOrgDEK unwraps the org DEK for a single (orgID, userID) using a pre-fetched
// OrgKeyMemberRecord and the user's salt (both provided by the caller to avoid per-org
// DB round-trips). Caches the result under OrgCacheKey(orgID) with the given TTL.
// Returns nil if record is nil (user has no key for this org).
func (s *OrgKeyService) UnlockOrgDEK(ctx context.Context, record *OrgKeyMemberRecord, userSalt []byte, password []byte, ttl time.Duration) error
```

Implementation:
1. `record == nil` → return nil (not an admin yet, or pending handshake)
2. `adminKEK = DeriveKEK(password, userSalt, orgKEKInfo)`; `defer zeroBytes(adminKEK)`
3. `orgDEK = UnwrapDEK(adminKEK, record.WrappedDEK)` — error → log Warn, return `ErrOrgKeyStale`
4. `cache.CacheDEK(ctx, OrgCacheKey(record.OrgID), orgDEK, ttl)`

#### `UnlockAllOrgDEKs`

Called by the login path (US-11.3). Batch-fetches all org key records for the user in one DB query, then unlocks each. The user's `user_keys.salt` is passed in — it is already in memory during login (used for `UnlockDEK`), eliminating the need for per-org `GetUserSalt` calls.

```go
// UnlockAllOrgDEKs fetches all org_key_members rows for userID in a single DB query,
// then calls UnlockOrgDEK for each. userSalt is the salt from user_keys — already
// available in the login flow, passed in to avoid a redundant DB query.
// Non-fatal per org: a stale wrap (wrong KEK) is logged as Warn and skipped.
// Returns nil even if some orgs fail — login must not be blocked by org DEK failures.
func (s *OrgKeyService) UnlockAllOrgDEKs(ctx context.Context, userID string, userSalt []byte, password []byte, ttl time.Duration) error
```

Implementation:
1. `records, err = store.GetOrgKeyMembersForUser(ctx, userID)` — DB error → log Warn, return nil (non-fatal)
2. For each record: `UnlockOrgDEK(ctx, record, userSalt, password, ttl)` — per-org error → log Warn, continue

#### `WrapOrgDEKForNewAdmin`

Called during the key handshake (Step 2 of US-11.5). The org DEK must already be in the cache (an existing admin is active).

```go
// WrapOrgDEKForNewAdmin retrieves the org DEK from cache and wraps it for a new admin.
// Returns ErrOrgDEKUnavailable if the org DEK is not currently cached
// (no admin is logged in — caller should return 409).
func (s *OrgKeyService) WrapOrgDEKForNewAdmin(ctx context.Context, orgID, newAdminUserID string, newAdminPassword []byte) error
```

Implementation:
1. `orgDEK = cache.GetDEK(ctx, OrgCacheKey(orgID))` — nil → return `ErrOrgDEKUnavailable`
2. `newAdminSalt = store.GetUserSalt(ctx, newAdminUserID)`
3. `newAdminKEK = DeriveKEK(newAdminPassword, newAdminSalt, orgKEKInfo)`; `defer zeroBytes(newAdminKEK)`
4. `wrappedDEK = WrapDEK(newAdminKEK, orgDEK)`
5. `existingRecord = store.GetOrgKeyMember(ctx, orgID, newAdminUserID)`
6. `store.UpsertOrgKeyMember(ctx, &OrgKeyMemberRecord{..., KeyVersion: existingVersion+1 or 1})`

#### `RewrapOrgDEKForAdmin`

Called when an admin changes their password (US-11.10). The current org DEK is available in cache (the user is mid-session).

```go
// RewrapOrgDEKForAdmin re-wraps the org DEK for a single admin with a new KEK derived
// from the new password.
//
// CONTRACT: This method is strictly cache-dependent. It reads the org DEK from
// cache.GetDEK(ctx, OrgCacheKey(orgID)). If the org DEK is not in cache, it returns
// ErrOrgDEKUnavailable — there is NO fallback using the old password, because the old
// password is not available in the password-change flow. The caller must handle
// ErrOrgDEKUnavailable as a non-fatal condition (log Warn, continue) and inform the
// user to trigger RotateOrgDEK via POST /orgs/:id/rotate-key.
//
// This is intentional: attempting to derive the old KEK from "somewhere" would require
// storing or re-requesting the old password, which violates the zero-knowledge model.
func (s *OrgKeyService) RewrapOrgDEKForAdmin(ctx context.Context, orgID, userID string, newPassword []byte) error
```

#### `RewrapAllOrgDEKsForAdmin`

Called by `RotateKeyHandler.ChangePassword` in US-11.10. Batch re-wraps across all admin orgs using a single DB query.

```go
// RewrapAllOrgDEKsForAdmin re-wraps the org DEK for every org this user admins.
// Uses GetOrgKeyMembersForUser (single batch DB query), then calls RewrapOrgDEKForAdmin
// per org. Non-fatal per org: ErrOrgDEKUnavailable and other errors are logged as Warn
// and skipped. Returns nil always — ChangePassword has already succeeded; callers must
// not receive a failure from this method.
func (s *OrgKeyService) RewrapAllOrgDEKsForAdmin(ctx context.Context, userID string, newPassword []byte) error
```

#### `RotateOrgDEK`

See US-11.9 for the full implementation. Signature defined here so callers can reference it.

```go
// OrgCredentialReEncryptor is the minimal interface RotateOrgDEK needs — it must
// re-encrypt org-owned provider_credentials rows atomically. Using this narrow
// interface rather than the full CredentialStore satisfies Interface Segregation:
// OrgKeyService does not need workspace-binding or auto-apply logic.
type OrgCredentialReEncryptor interface {
    // ReEncryptOrgCredentials re-encrypts all provider_credentials where
    // owner_type='org' AND owner_id=orgID, from oldDEK to newDEK, atomically.
    // Returns the count of rows updated.
    ReEncryptOrgCredentials(ctx context.Context, orgID string, oldDEK, newDEK []byte) (int, error)
}

// RotateOrgDEK generates a new org DEK and re-wraps it for the rotating admin.
// All other admins have their org_key_members rows deleted and pending_key_wrap
// set to true — they must call accept-key to re-wrap with the new DEK.
//
// The new org DEK is cached under OrgCacheKey(orgID) with a 7-day TTL (not the
// session TTL). This extended TTL is intentional: other admins must be able to call
// accept-key after the rotating admin's session expires. Without the extended TTL,
// a single-admin org becomes permanently deadlocked if the rotating admin logs out
// before other admins complete their accept-key calls.
//
// adminUserID and adminPassword identify the rotating admin — their KEK is derived
// to re-wrap the new org DEK for them. The org DEK must also be in cache (to
// re-encrypt credentials); ErrOrgDEKUnavailable is returned if not.
//
// orgMemberUpdater is a minimal interface for setting pending_key_wrap on other admins.
// reEncryptor handles credential re-encryption (see OrgCredentialReEncryptor above).
func (s *OrgKeyService) RotateOrgDEK(
    ctx context.Context,
    orgID, adminUserID string,
    adminPassword []byte,
    reEncryptor OrgCredentialReEncryptor,
    orgMemberUpdater OrgMemberPendingUpdater,
) error
```

Supporting interface:
```go
// OrgMemberPendingUpdater sets pending_key_wrap on all non-rotating admins after rotation.
type OrgMemberPendingUpdater interface {
    SetPendingKeyWrapForOtherAdmins(ctx context.Context, orgID, excludeUserID string) error
}
```

#### `GetOrgDEK`

Used by `decryptBinding` to retrieve the cached org DEK.

```go
// GetOrgDEK returns the cached org DEK. Returns ErrOrgDEKUnavailable if not in cache.
func (s *OrgKeyService) GetOrgDEK(ctx context.Context, orgID string) ([]byte, error)
```

### Sentinel errors

```go
var (
    ErrOrgDEKUnavailable = errors.New("org DEK not currently cached — an org admin must be logged in")
    ErrOrgKeyNotFound    = errors.New("org key member record not found")
    ErrOrgKeyStale       = errors.New("org key member record is stale — org DEK rotation required")
)
```

## Acceptance Criteria

- `InitializeOrgKeys` generates a unique DEK per call, stores exactly one `org_key_members` row, returns plaintext DEK to caller
- `UnlockOrgDEK` with correct password → DEK in cache under `"org:<orgID>"`, returns nil
- `UnlockOrgDEK` with wrong password → returns error; nothing written to cache
- `UnlockOrgDEK` with no `org_key_members` row → returns nil (not an admin)
- `WrapOrgDEKForNewAdmin` with DEK not in cache → returns `ErrOrgDEKUnavailable`
- `WrapOrgDEKForNewAdmin` with DEK in cache → creates new `org_key_members` row; new admin can subsequently `UnlockOrgDEK`
- `RewrapOrgDEKForAdmin` → updates `org_key_members` row; old KEK no longer unwraps
- All functions zero sensitive key material with `zeroBytes` before returning
- `PgOrgKeyStore` correctly reads/writes `org_key_members` and reads `user_keys.salt` via `GetUserSalt`
