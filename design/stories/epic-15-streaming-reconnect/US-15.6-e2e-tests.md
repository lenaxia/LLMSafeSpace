**Epic:** 15 - Streaming State Resilience & Mid-Stream Reconnect
**Priority:** High
**Depends on:** US-15.1, US-15.2, US-15.3, US-15.4, US-15.5

## User Story

As a developer maintaining the streaming reconnect feature, I need comprehensive automated tests that verify the full reconnect flow works correctly, so that future changes don't silently break the UX.

## Acceptance Criteria

- [ ] Unit tests for the boundary detection logic (historyPartIds gate)
- [ ] Unit tests for the state priority logic (localStreaming vs serverBusy)
- [ ] Integration tests simulating the full reconnect scenario (mount → status poll → history fetch → SSE events → idle reconciliation)
- [ ] Integration tests for the SSE reconnect edge case (disconnect while busy, reconnect, status re-poll)
- [ ] Tests verify no duplicate rendering in any scenario
- [ ] Tests verify no content loss in any scenario
- [ ] Tests use the existing test infrastructure (`vitest`, `@testing-library/react`, mock SSE via the existing `useEventStream` mock pattern)
- [ ] All existing `ChatPage.sse.test.tsx` tests still pass (no regressions)

### Mock Helpers

```typescript
function mockBusySession(sessionId: string) {
  (workspacesApi.getStatus as ReturnType<typeof vi.fn>).mockResolvedValue({
    phase: "Active",
    sessions: [{ id: sessionId, status: "busy" }],
  });
}

function mockIdleSession(sessionId: string) {
  (workspacesApi.getStatus as ReturnType<typeof vi.fn>).mockResolvedValue({
    phase: "Active",
    sessions: [{ id: sessionId, status: "idle" }],
  });
}

function mockHistoryWithParts(parts: Array<{ id: string; type: string; text: string }>) {
  (messagesApi.getHistory as ReturnType<typeof vi.fn>).mockResolvedValue([
    {
      id: "msg-1",
      role: "assistant",
      parts: parts.map(p => ({ id: p.id, type: p.type, text: p.text })),
    },
  ]);
}

function makePartUpdatedWithId(sessionId: string, partId: string, text: string): WorkspaceStreamEvent {
  return {
    type: "opencode.event",
    event_type: "message.part.updated",
    data: {
      type: "message.part.updated",
      properties: {
        sessionID: sessionId,
        part: { id: partId, sessionID: sessionId, messageID: "msg-2", type: "text", text },
      },
    },
  } as unknown as WorkspaceStreamEvent;
}

function makePartDeltaWithId(sessionId: string, partId: string, delta: string): WorkspaceStreamEvent {
  return {
    type: "opencode.event",
    event_type: "message.part.delta",
    data: {
      type: "message.part.delta",
      properties: {
        sessionID: sessionId,
        messageID: "msg-2",
        partID: partId,
        field: "text",
        delta,
      },
    },
  } as unknown as WorkspaceStreamEvent;
}
```
