# US-24.11: Prometheus Metrics

**Domain:** Observability
**Depends On:** US-24.5
**Priority:** Medium

## Summary

Add Prometheus metrics for failure classification, recovery attempts, safe mode, and degraded state. No per-workspace labels (F18 — cardinality safe at scale).

## Metrics

```go
// Counters
var (
    FailureClassificationsTotal = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_failure_classifications_total",
            Help: "Failure classifications by class and raw K8s reason",
        },
        []string{"class", "pod_reason", "container_reason"},
    )

    FailureClassificationUnknownTotal = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_failure_classification_unknown_total",
            Help: "Classifications that hit the default/unknown path",
        },
        []string{"pod_reason", "container_reason"},
    )

    RecoveryAttemptsTotal = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_recovery_attempts_total",
            Help: "Recovery attempts by class and outcome",
        },
        []string{"failure_class", "outcome"}, // outcome: retry|safe_mode|recovered
    )

    ControllerRestartsTotal = prometheus.NewCounter(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_controller_restarts_total",
            Help: "Health-check initiated pod restarts",
        },
    )

    SafeModeEntriesTotal = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_safe_mode_entries_total",
            Help: "Safe mode entries by trigger",
        },
        []string{"trigger"},
    )

    SafeModeExitsTotal = prometheus.NewCounterVec(
        prometheus.CounterOpts{
            Name: "llmsafespace_workspace_safe_mode_exits_total",
            Help: "Safe mode exits by method",
        },
        []string{"method"},
    )
)

// Gauges
var (
    WorkspacesInRecoveryTotal = prometheus.NewGauge(prometheus.GaugeOpts{
        Name: "llmsafespace_workspaces_in_recovery_total",
        Help: "Workspaces currently in backoff (phase=Creating + NextRetryAt set)",
    })

    WorkspacesInSafeModeTotal = prometheus.NewGauge(prometheus.GaugeOpts{
        Name: "llmsafespace_workspaces_in_safe_mode_total",
        Help: "Workspaces currently in safe mode",
    })

    WorkspacesDegradedTotal = prometheus.NewGauge(prometheus.GaugeOpts{
        Name: "llmsafespace_workspaces_degraded_total",
        Help: "Workspaces with any Degraded condition",
    })
)

// Histograms
var (
    RecoveryDurationSeconds = prometheus.NewHistogramVec(
        prometheus.HistogramOpts{
            Name:    "llmsafespace_workspace_recovery_duration_seconds",
            Help:    "Time from failure to Active recovery",
            Buckets: []float64{5, 10, 30, 60, 120, 300, 600, 1800, 3600},
        },
        []string{"failure_class"},
    )

    TimeToReadySeconds = prometheus.NewHistogram(prometheus.HistogramOpts{
        Name:    "llmsafespace_workspace_time_to_ready_seconds",
        Help:    "Pod creation to readiness probe pass",
        Buckets: []float64{5, 10, 15, 30, 60, 120, 300},
    })

    BackoffAppliedSeconds = prometheus.NewHistogramVec(
        prometheus.HistogramOpts{
            Name:    "llmsafespace_workspace_backoff_applied_seconds",
            Help:    "Backoff durations applied",
            Buckets: []float64{5, 10, 20, 40, 60, 120, 300, 600},
        },
        []string{"failure_class"},
    )
)
```

## Emission Points

| Metric | Where emitted |
|---|---|
| `failure_classifications_total` | `classifyFailure()` |
| `failure_classification_unknown_total` | `classifyFailure()` default branch |
| `recovery_attempts_total` | `enterRecovery()` |
| `controller_restarts_total` | `checkAgentHealth()` on threshold trip |
| `safe_mode_entries_total` | `enterRecovery()` when SafeMode triggered |
| `safe_mode_exits_total` | `restartGeneration` handler, `handleResuming` |
| `workspaces_in_recovery_total` | Reconcile loop (count workspaces with NextRetryAt set) |
| `workspaces_in_safe_mode_total` | Reconcile loop (count workspaces with SafeMode=true) |
| `workspaces_degraded_total` | Reconcile loop (count workspaces with Degraded condition) |
| `recovery_duration_seconds` | Transition from Creating→Active (if ConsecutiveFailures > 0) |
| `time_to_ready_seconds` | Transition from Creating→Active (StartTime delta) |
| `backoff_applied_seconds` | `enterRecovery()` |

## Acceptance Criteria

1. All metrics registered in `SetupMetrics()`
2. No per-workspace labels on any metric (F18)
3. `failure_classification_unknown_total` fires on default path
4. Gauge metrics updated on every reconcile (not just on state change)
5. Histograms use appropriate bucket ranges
