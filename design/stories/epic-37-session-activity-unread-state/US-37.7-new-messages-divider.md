# US-37.7: "New Messages" Divider in MessageList

**Epic:** 37 - Session Activity & Unread State UX
**Priority:** High
**Depends on:** US-37.3 (last_seen_at in session data)

## User Story

As a user navigating to a session with new responses, I need a visual divider showing where new content begins, so I can quickly find what changed since my last visit.

## Acceptance Criteria

- [ ] Divider rendered at the boundary between seen and unseen messages
- [ ] No divider when `lastSeenAt` is null (first visit)
- [ ] No divider when caught up (`lastSeenAt >= lastMessageAt`)
- [ ] Divider is a horizontal line with centered "New messages" text
- [ ] Accessible (`role="separator"`, `aria-label`)
- [ ] No crash when messages lack `createdAt`

## Assumptions

| # | Assumption | Validation |
|---|-----------|-----------|
| A1 | Messages sorted chronologically by `selectChronological` | ✅ `messages.ts` |
| A2 | Each message has `createdAt?: string` (ISO) | ✅ `api/types.ts:152` |
| A3 | Newest messages are on the first page (loaded by default) | ✅ `getHistoryPage` fetches newest first |
| A4 | `createdAt` and `last_seen_at` may differ by up to ~1s due to clock skew | ✅ D3 — 1s buffer |

## Implementation

### Prop Chain

```tsx
// ChatPage → ChatView → MessageList
const currentSession = sessions?.find(s => s.id === sessionId);
<MessageList lastSeenAt={currentSession?.lastSeenAt} ... />
```

### Divider Computation

```typescript
const CLOCK_SKEW_BUFFER_MS = 1000;

const dividerIndex = useMemo(() => {
  if (!lastSeenAt) return -1;
  const threshold = new Date(lastSeenAt).getTime() - CLOCK_SKEW_BUFFER_MS;
  return messages.findIndex(
    (msg) => msg.createdAt && new Date(msg.createdAt).getTime() > threshold
  );
}, [messages, lastSeenAt]);
```

The 1s buffer means messages within 1 second before `lastSeenAt` are treated as "seen" — handles clock skew between API server and agent pod.

### Rendering

```tsx
{messages.map((msg, i) => (
  <React.Fragment key={msg.id}>
    {i === dividerIndex && (
      <div role="separator" aria-label="New messages"
           className="flex items-center gap-2 py-2 my-1">
        <div className="flex-1 border-t border-border" />
        <span className="text-xs text-muted-foreground font-medium whitespace-nowrap px-2">
          New messages
        </span>
        <div className="flex-1 border-t border-border" />
      </div>
    )}
    <MemoizedBubble message={msg} ... />
  </React.Fragment>
))}
```
