# US-14.5: Go SDK

**Epic:** 14 - Multi-Language SDKs & VS Code Extension
**Priority:** High
**Depends on:** US-14.1

## User Story

As a Go developer building infrastructure tooling or Kubernetes operators, I want a typed Go client for LLMSafeSpace, so that I can integrate sandbox management into Go services with idiomatic error handling and context propagation.

## Acceptance Criteria

- [ ] Module `github.com/lenaxia/llmsafespace/sdk/go` exists in `sdks/go/`
- [ ] All REST endpoints have typed methods accepting `context.Context`
- [ ] Authentication: API key and credential-based (auto JWT refresh)
- [ ] Errors implement `error` interface with typed assertions (`errors.As`)
- [ ] Uses `net/http` standard library (no third-party HTTP client dependency)
- [ ] Follows Go conventions: `(value, error)` returns, option pattern for config
- [ ] Generated types match OpenAPI spec
- [ ] `go test ./...` passes
- [ ] README with usage examples
- [ ] Module tagged and importable via `go get`

## Assumptions

| # | Assumption | Validation |
|---|-----------|-----------|
| A1 | `oapi-codegen` supports OpenAPI 3.1 | ✅ oapi-codegen v2 supports 3.1 natively |
| A2 | Separate Go module avoids pulling K8s deps for SDK users | ✅ Correct — SDK module has its own go.mod with zero K8s imports |
| A3 | `POST /workspaces/:id/sessions/:sessionId/message` blocks until LLM responds | ✅ Verified: synchronous proxy. May take 30-120s. |

## Technical Details

**Generation approach:**

Use `oapi-codegen` with the `client` and `types` generation targets.

```bash
# In sdks/Makefile
generate-go:
    oapi-codegen -generate types -package llmsafespace -o sdks/go/types.gen.go sdks/openapi.yaml
    oapi-codegen -generate client -package llmsafespace -o sdks/go/client.gen.go sdks/openapi.yaml
```

**Package structure:**

```
sdks/go/
├── go.mod                    # github.com/lenaxia/llmsafespace/sdk/go
├── go.sum
├── client.go                 # High-level client facade (hand-written)
├── auth.go                   # Auth management (token refresh)
├── errors.go                 # Typed error types
├── options.go                # Functional options for client config
├── types.gen.go              # Generated types (DO NOT EDIT)
├── client.gen.go             # Generated low-level client (DO NOT EDIT)
├── client_test.go            # Unit tests

└── README.md
```

**Public API surface:**

```go
package main

import (
    "context"
    "fmt"
    llm "github.com/lenaxia/llmsafespace/sdk/go"
)

func main() {
    ctx := context.Background()

    // Initialize with API key
    client, err := llm.NewClient(
        llm.WithBaseURL("https://llmsafespace.example.com"),
        llm.WithAPIKey("lsp_..."),
    )
    if err != nil {
        panic(err)
    }

    // Or with credentials
    client, err := llm.NewClient(
        llm.WithBaseURL("https://llmsafespace.example.com"),
        llm.WithCredentials("user@example.com", "password"),
    )

    // Workspace operations
    ws, err := client.Workspaces.Create(ctx, llm.CreateWorkspaceRequest{
        Name:        "my-project",
        Runtime:     "python:3.11",
        StorageSize: "10Gi",
    })

    list, err := client.Workspaces.List(ctx, llm.ListOptions{Limit: 10})
    status, err := client.Workspaces.GetStatus(ctx, ws.ID)
    err = client.Workspaces.Suspend(ctx, ws.ID)
    err = client.Workspaces.Resume(ctx, ws.ID)
    err = client.Workspaces.Activate(ctx, ws.ID)

    // Session operations (EnsureSession activates workspace + creates session)
    session, err := client.Sessions.Ensure(ctx, ws.ID)
    // SendMessage blocks until LLM responds. Use context with timeout for control.
    msgCtx, cancel := context.WithTimeout(ctx, 120*time.Second)
    defer cancel()
    resp, err := client.Sessions.SendMessage(msgCtx, ws.ID, session.SessionID, "Write hello world")
    history, err := client.Sessions.GetHistory(ctx, ws.ID, session.SessionID)
}
```

**Error handling:**

```go
import "errors"

ws, err := client.Workspaces.Get(ctx, "nonexistent")
if err != nil {
    var notFound *llm.NotFoundError
    var authErr *llm.AuthError
    var rateLimited *llm.RateLimitError

    switch {
    case errors.As(err, &notFound):
        fmt.Printf("not found: %s\n", notFound.Message)
    case errors.As(err, &rateLimited):
        fmt.Printf("retry after: %s\n", rateLimited.RetryAfter)
    case errors.As(err, &authErr):
        fmt.Println("re-authenticate")
    default:
        fmt.Printf("unexpected: %v\n", err)
    }
}
```

**Design decisions:**

- Separate Go module (not part of the main `github.com/lenaxia/llmsafespace` module) to avoid pulling in K8s dependencies for SDK users
- Uses `net/http` with a configurable `http.Client` (users can inject their own transport for proxies, retries, etc.)
- Context propagation on every method (standard Go pattern)
- Functional options pattern for client configuration

**Dependencies:**

- None beyond Go standard library (net/http, encoding/json, context)

## Design Reference

- `sdks/openapi.yaml` (US-14.1)
- `pkg/types/types.go` — canonical type definitions (SDK types are generated separately to avoid importing K8s deps)
