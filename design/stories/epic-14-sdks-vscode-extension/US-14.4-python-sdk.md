# US-14.4: Python SDK

**Epic:** 14 - Multi-Language SDKs & VS Code Extension
**Priority:** High
**Depends on:** US-14.1

## User Story

As a Python developer building AI agents, I want a typed Python SDK for LLMSafeSpace, so that I can programmatically create sandboxes and interact with agents from scripts, notebooks, and agent frameworks.

## Acceptance Criteria

- [ ] Package `llmsafespace` exists in `sdks/python/`
- [ ] All REST endpoints have typed methods with dataclass/Pydantic models
- [ ] Supports both sync (`httpx`) and async (`httpx.AsyncClient`) usage
- [ ] Authentication: API key and credential-based (auto JWT refresh)
- [ ] Typed errors with `LLMSafeSpaceError` hierarchy
- [ ] Python 3.10+ (type hints, `|` union syntax)
- [ ] Zero heavy dependencies (httpx + pydantic only)
- [ ] Generated models match OpenAPI spec
- [ ] README with usage examples
- [ ] `pytest` passes (unit + integration tests)
- [ ] `pyproject.toml` configured for PyPI publishing

## Technical Details

**Generation approach:**

Use `openapi-python-client` to generate the initial client, then wrap with a hand-written facade for ergonomics.

```bash
# In sdks/Makefile
generate-python:
    openapi-python-client generate --path sdks/openapi.yaml --output-path sdks/python/src/llmsafespace/generated
```

**Public API surface:**

```python
from llmsafespace import LLMSafeSpace, AsyncLLMSafeSpace

# Sync client
client = LLMSafeSpace(
    base_url="https://llmsafespace.example.com",
    api_key="lsp_...",
)

# Async client
client = AsyncLLMSafeSpace(
    base_url="https://llmsafespace.example.com",
    api_key="lsp_...",
)

# Workspace operations
workspace = client.workspaces.create(name="my-project", runtime="python:3.11", storage_size="10Gi")
workspaces = client.workspaces.list()
status = client.workspaces.get_status(workspace.id)
client.workspaces.suspend(workspace.id)
client.workspaces.resume(workspace.id)
client.workspaces.activate(workspace.id)

# Session operations (ensure_session activates workspace + creates session)
session = client.sessions.ensure(workspace.id)
# send_message blocks until LLM responds (configurable timeout, default 120s)
response = client.sessions.send_message(workspace.id, session.session_id, content="Write hello world")
history = client.sessions.get_history(workspace.id, session.session_id)

# Context manager for auto-cleanup
with client.workspaces.create(name="temp", runtime="python:3.11") as ws:
    session = client.sessions.ensure(ws.id)
    result = client.sessions.send_message(ws.id, session.session_id, content="Run tests")
    # workspace suspended on exit (not terminated — preserves state)
```

**Error handling:**

```python
from llmsafespace import LLMSafeSpaceError, NotFoundError, AuthError, RateLimitError

try:
    client.workspaces.get("nonexistent")
except NotFoundError as e:
    print(f"Not found: {e.message}")
except RateLimitError as e:
    print(f"Rate limited, retry after: {e.retry_after}")
except LLMSafeSpaceError as e:
    print(f"API error {e.status}: {e.message}")
```
