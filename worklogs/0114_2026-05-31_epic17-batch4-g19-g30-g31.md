## G19 — mise attestations

`runtimes/base/Dockerfile`: flipped `MISE_GITHUB_ATTESTATIONS=0`
→ `1`. Mise now enforces Sigstore-backed GitHub attestation checks
on every download. Build environments with restricted egress need
to allow Sigstore + GitHub OIDC.

## G30 — Cilium FQDN docs

Standard k8s NetworkPolicy cannot match FQDNs. The chart-wide
NetPol uses CIDR-based rules. Operators wanting FQDN-level egress
control (api.openai.com only, etc.) should deploy a
CiliumNetworkPolicy or Calico GlobalNetworkPolicy alongside the
chart. NOTES.txt now contains a worked example.

## G31 — Frontend security headers

`charts/llmsafespace/values.yaml::frontend.ingress.annotations`:
default annotations now set CSP, X-Frame-Options DENY, X-Content-
Type-Options nosniff, Referrer-Policy strict-origin-when-cross-origin,
and HSTS via the ingress-nginx `more_set_headers` mechanism.
Operators using Traefik/HAProxy override with their controller's
equivalent annotation.

CSP `frame-ancestors 'none'` + X-Frame-Options DENY = double-block
on clickjacking. CSP `default-src 'self'` constrains XSS blast
radius (RT-7.9).
