## Changes

### G11 — Pod Security Admission

`charts/llmsafespace/templates/namespace.yaml` adds PSA labels at
all three levels (enforce, audit, warn) when
`namespace.podSecurityEnforce` is set (default `"restricted"`).

`charts/llmsafespace/values.yaml`:
```yaml
namespace:
  podSecurityEnforce: "restricted"
  podSecurityVersion: "latest"
```

### G12 — Proxy timeout

`api/internal/handlers/proxy.go`: default
`ResponseHeaderTimeout` lowered from `300s` → `60s`. Comment
documents that streaming endpoints bypass this client.

### G22 — EnableServiceLinks

`controller/internal/workspace/controller.go`: workspace pod spec
now sets `EnableServiceLinks: ptr(false)`.

### G23 — PVC mountOptions

`charts/llmsafespace/templates/NOTES.txt` adds a section explaining
that operators should configure their workspace StorageClass with
`mountOptions: [nosuid, nodev]`. The chart can't ship a StorageClass
because it doesn't know the operator's CSI provisioner.

### G24 — seccompProfile

`controller/internal/workspace/controller.go`:
`buildPodSecurityContext` now sets
`SeccompProfile: { Type: RuntimeDefault }`.

### G27 — Login timing

`api/internal/services/auth/auth.go::Login`: when `user == nil` OR
db error, run a dummy `bcrypt.CompareHashAndPassword` against a
real bcrypt hash so total response time matches the wrong-password
branch. Mirrors the existing pattern in `VerifyPassword`.
